## 集成方式

### background-csp-bypass.js

```javascript
// 加载自动生成的规则
async function loadAutoGeneratedRules() {
  const rulesModule = await import('./background-dnr-rules-auto.js')
  AUTO_GENERATED_JS_REDIRECT_RULES = rulesModule.AUTO_GENERATED_JS_REDIRECT_RULES
}

// 注册规则
async function registerJSRedirectRules() {
  await loadAutoGeneratedRules()

  const existingRules = await chrome.declarativeNetRequest.getDynamicRules()
  const oldIds = existingRules
    .filter(rule => rule.id >= 4000 && rule.id < 5000)
    .map(rule => rule.id)

  if (oldIds.length > 0) {
    await chrome.declarativeNetRequest.updateDynamicRules({ removeRuleIds: oldIds })
  }

  await chrome.declarativeNetRequest.updateDynamicRules({
    addRules: AUTO_GENERATED_JS_REDIRECT_RULES
  })
}
```

## 维护方式

### 添加新规则

1. 编辑 `scripts/generate-dnr-rules.js`
2. 在 `JS_CDN_MAP` 中添加新库配置
3. 运行 `node scripts/generate-dnr-rules.js`
4. 重新加载扩展

### 更新版本

```javascript

  file: 'lodash.min.js',
  defaultVersion: '4.17.21', // 更新为新版本
  // ...
}
```
